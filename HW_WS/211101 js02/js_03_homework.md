#### 2. JavaScript에서 동기와 비동기 함수의 차이점을 서술하시오

* 동기 
  * 순차적, 직렬적 Task 수행
  * 요청을 보낸 후 응답을 받아야만 다음 동작이 이루어진다.
  * 즉, 한번에 하나의 요청을 처리 할 수 있다
* 비동기
  * 병렬적 Task 수행
  * 요청을 보내고 응답을 기다리지 않고 다음 코드가 실행된다.
  * 즉, 동시에 여러 요청을 처리할 수 있다.
